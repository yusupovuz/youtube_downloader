# YouTube Video Downloader API

Этот проект разработан в качестве экзаменационного задания. С помощью `yt-dlp` и `FastAPI` он позволяет скачивать видео в фоновом режиме (threading), кэшировать файлы и отслеживать прогресс загрузки.

## Как запустить?

1. Скачайте проект и перейдите в его папку.
2. Создайте и активируйте виртуальное окружение:
   ```bash
   python -m venv venv
   source venv/bin/activate  # Для Mac/Linux
   venv\Scripts\activate     # Для Windows
